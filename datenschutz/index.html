2. Datenverarbeitung auf dieser Website

Beim Besuch unserer Website werden keine personenbezogenen Daten gespeichert oder ausgewertet. Es werden keine Cookies gesetzt, kein Tracking und keine Analyse-Tools verwendet. Das Hosting erfolgt über GitHub Pages.
